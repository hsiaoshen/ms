1. 获取到页面所有checkbox怎么做?

```js
//使用jquery
var checkbox = $(':checkbox');

// 使用js原生

    var input = document.querySelectorAll('input');
    var checkboxArray = [];
    for(let i = 0; i < input.length; i++){
          var obj = input[i];
          if(obj.type == 'checkbox'){
              checkboxArray.push(obj);
          }
    }
```
